# Badane elementy

## Ogólne UI

- [ ] Porównanie czytelności obu aplikacji webowych np poprzez mierzenie czasu w zidentyfikowaniu konkretnej części
- [ ] Porównanie użytych kolorów i efektu jaki one dają

## Zarządzanie wiadomościami

- [ ] Czytelność i obsługa paska bocznego (odebrane, wysłane ...) - łatwość wykonania pożądanej czynności
- [ ] Wyświetlanie wiadomości w konkretnej kategorii - UI, czytelność
- [ ] Filtrowanie - łatwość filtrowania według danego kryterium, obsługa elementu UI odpowiadającego za filtrowanie
- [ ] Pozostałe funkcjonalności, porównanie ilości oraz czytelności pozostałych elementów UI oraz ich przeznaczenia
- [ ] Zmierzenie czasu jaki zajmie oznaczenie jako spam

## Wysyłanie wiadomości

- [ ] Porównanie czasu jaki zajmie znalezienie przycisku do utworzenia wiadomości
- [ ] Zmierzenie ogólnego czasu jaki zajmie wysłanie wiadomości (nie wliczając czasu pisania wiadomości)
- [ ] Zmierzenie czasu znalezienia przycisku odpowiedzi na wiadomość w różnych scenariuszach (np. z poziomu widoku odebranych, z poziomu widoku danej wiadomości)


## Kalendarz

- [ ] Zmierzenie czasu jaki zajmie znalezienie i uruchomienie kalendarza
- [ ] Zmierzenie czasu jaki zajmie dodanie konkretnego wydarzenia w kalendarzu od-do
- [ ] Zmierzenie czasu jaki zajmie 


## Ogólne odczucia

- [ ] Zebranie odpowiedzi osób badanych na temat czytelności UI obu aplikacji
- [ ] Zebranie odpowiedzi na temat łatwości obsługi danych elementów
